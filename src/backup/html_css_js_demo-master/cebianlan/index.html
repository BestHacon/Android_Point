欢迎光临仵老大科技后台管理系统
从前，有个王后，她很想生一个孩子，就天天向上帝祈祷。 有一天她正在祈祷时，有只青蛙跳出来说：“你的愿望要实现了，很快你就会有孩子。” 果然，九个月后，王后生下了一个非常漂亮可爱的小公主。 王后和国王都高兴极了。在小公主满月时，他们举行了一个盛大的宴会庆祝。 他们给13个女巫也发去邀请信，但是由于信差的疏忽，有一个女巫没收到邀请信，她十分生气。 在宴会上，女巫们都纷纷献上对小公主的祝福，比如“美丽”、“聪明”等等。 当轮到最后一位女巫祝福时，那位没收到邀请信的女巫却走进来，恶狠狠地说：“我祝公主在十五岁那年碰到纺锤而死。” 说完她就转身离开了。
大家都大吃一惊，因为其他女巫也不能解开这个咒语。 现在只有一位女巫还没祝福，大家对她说：“咒语不能解开，怎样挽救公主就看你了！” 这位女巫说：“我祝公主没有死，她只是睡了，当有英俊、善良、聪明的王子来唤醒她的时候，她就会醒来。” 宴会结束后，国王十分担心，他命令全国把所有的纺锤都烧了，以后不许有这种东西出现。 小公主渐渐长大成人，就像12个女巫祝福的那样，她漂亮、聪明，性格很好，所有看过她的人都很喜欢她。 公主十五岁的时候，有一天，她在宫里到处玩耍，忽然看到一个老太婆手拿纺锤在纺线。 公主从来没见过这东西，十分好奇。 老太婆主动把纺锤递给她看。 就在公主碰到纺锤的一瞬间，她倒在了地上。女巫的咒语果然实现了。 这个老太婆其实就是那个下咒语的女巫。 就在公主倒下的时候，整个王宫也好像凝固了。 国王和王后正在吃水果，他们刚把水果放在嘴边就不动了。 正在泡茶的宫女也不动了，茶壶里倒出的水也停在空中。 王宫的一切都静止了。 很快王宫周围长出玫瑰花围成的篱笆，玫瑰花篱笆把王宫完全围住了。 如果不说，都没人知道里面有王宫。 但是这个故事一直流传着，大家都知道这个玫瑰花篱笆里面有个王宫，王宫里有位美丽聪明的公主，等待一位同等英俊聪明的王子去唤醒她。 许多别国的王子都尝试过进入王宫，他们想见到美丽的公主，也都自信自己的英俊和聪明能唤醒公主。 可是，他们完全进不了玫瑰花篱笆，个个都弄得遍体鳞伤，十分狼狈。 就这样一百年过去了，也不知道有多少位王子来过。 有一天，又来了位王子，当他听说这个传说后，也想进入王宫。 大家都极力劝阻他，但他不理会。
奇怪的是，这次玫瑰花篱笆没伤害他，反而给他主动让道，他走到哪里，哪里的玫瑰花就让开了。 就这样他顺利进入王宫。 他看到了正在吃水果的国王和王后，还有在泡茶的宫女，他们都像雕像一样一动不动。 最后他看到了倒在床上的公主。 公主比他想象的还要美丽。 王子亲了亲她的额头。 奇迹发生了，公主坐了起来，就好像她刚睡醒一样。 这时王宫也好像被解冻一样，国王和王后把水果放进嘴里，宫女继续把茶壶里的茶倒入杯里。 仿佛这一百年似乎只是一秒的事。 后来，公主和王子结了婚，他们同样善良、聪明、美丽，过得十分幸福。
